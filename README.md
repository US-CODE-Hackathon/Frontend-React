# 情마루
> AI 기반 정서 분석과 비동기 음성 소통을 통해  
> **고령층의 감정 표현과 디지털 자립을 지원하는 키오스크 서비스**입니다.
</div>

## Frontend Tech Stack

- **React**
- **TypeScript**
- **styled-components**

## 실행 방법

```bash
# 1. 의존성 설치
npm install

# 2. 개발 서버 실행
npm run dev
```

---

정마루는 따뜻한 기술로 디지털 소외를 줄이는 것을 목표로 합니다.
